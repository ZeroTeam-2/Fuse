# step-pages — delta (implement-file-step-upload)

## ADDED Requirements

### Requirement: Настройки файла у блока dropzone
Блок `dropzone` SHALL иметь опциональные настройки допустимых форматов файла (расширения/MIME-типы) и лимита размера в МБ. Инспектор свойств MUST показывать эти настройки только для блока `dropzone`. Страница исполнения (PageRunner) MUST валидировать выбранный файл по этим настройкам до начала загрузки; блоки без настроек ограничиваются только серверной проверкой типов и лимитов.

#### Scenario: Редактор задаёт ограничения файла
- **WHEN** редактор выбирает dropzone-блок и указывает в инспекторе допустимые форматы и максимальный размер
- **THEN** настройки сохраняются в модели блока, а на странице исполнения подпись dropzone отражает ограничения

#### Scenario: Файл вне ограничений отклоняется на клиенте
- **WHEN** пользователь на странице шага выбирает файл недопустимого формата или больше лимита блока
- **THEN** загрузка не начинается и отображается ошибка с указанием допустимых форматов/лимита
